WEB212 PHP & MySQL

{ Take Your Programming Skills to the Next Level! }

Did you know that Facebook, Wikipedia, and WordPress were all built on PHP and MySQL? PHP is the fastest-growing scripting language in the world, and MySQL is the most popular open source database. Now they're coming together in one class at MCC!

Building upon your server-side skills from WEB105 and your programming skills from PRG105, this course gives you the skills and knowledge to create a dynamic-driven website using PHP that links to a MySQL database. This course is centered on developing a working business-web application through hands-on exercises and projects.

Note: If you are pursuing the Web Development Certificate or the Web Design and Development degree, this course is replacing PRG148 and DBM110 and combining the content into one course.
